確信度肯定否定

2. must vs have to（否定の違い）

形意味例
must not禁止「〜してはならない」You must not run here. ここで走ってはいけない。
don't have to不要「〜する必要はない」You don't have to run. 走る必要はない。

3. can vs be able to

場面canbe able to
現在の一般的能力○ I can swim.○ I am able to swim.
過去の一般的能力○ I could swim.○ I was able to swim.
過去の具体的達成✕○ I was able to escape.（逃げることができた）
助動詞の後ろ✕○ I will be able to swim.

4. will vs be going to

willbe going to
その場の決定○ I'll help you.✕
以前からの予定△○ I'm going to visit Kyoto.
根拠のある予測△○ It's going to rain.（雲を見て）
単純な推測○ It will rain tomorrow.△

5. used to vs would（過去の習慣）

used towould
過去の習慣的動作○○
過去の状態○ I used to live in Tokyo.✕（状態動詞には使えない）
今との対比○（今はもう違う含意）△（対比は弱い）

6. 法助動詞 vs 準助動詞

特徴法助動詞準助動詞
形1語（can, must, will …）句（have to, be able to …）
後ろの動詞原形不定詞to不定詞（多くの場合）
否定助動詞 + notdo not + 動詞 / be + not …
疑問助動詞を前に出すDo/Does/Is … ?
時制変化限定的（can→could）自由（has to, had to, will have to …）
重ね使い✕ 2つ並べられない○ will have to, might be able to
